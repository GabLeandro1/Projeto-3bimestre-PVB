<h2>📌 Descrição do Projeto</h2>
<p>Este software foi desenvolvido durante o segundo ano do ensino médio para a disciplina Programação Visual Básica (PVB). Ele realiza o cálculo do valor de cada parcela de um produto, organiza as datas de pagamento e permite que o usuário efetue os pagamentos diretamente pelo sistema.</p>

---

<h2>🛠 Tecnologias Utilizadas</h2>
<ul>
<li>Linguagem: C#</li>
<li>Framework: .NET 4.8</li>
<li>IDE: Visual Studio 2017</li>
</ul>

---

<h2>🚀 Como Executar:</h2>
<p>Para executar o projeto, siga os passos abaixo:</p>
<ol>
<li>Abra o Visual Studio (versão compatível com .NET 4.8 ou superior);</li>
<li>Clique em "Abrir Projeto/Solução";</li>
<li>Selecione o arquivo "Projeto_Wagner_3bi.sln";</li>
<li>Compile e execute o projeto clicando em "Iniciar" ou pressionando F5.</li>
</ol>
